Privacy Policy

Live Checkout Friction Monitor

This policy describes the data handling model for the admin-only Ecwid dashboard deployed from this repository.

Effective date 2026-04-06 Applies to the current static deployment of the app.
Data model Direct Ecwid reads The app reads store data inside the Ecwid admin iframe using the runtime payload token.
Storage model Browser-local only Owner notes and dashboard preferences stay in the current browser.

What The App Processes

Operational Store Data

  • Ecwid admin data
    Store profile, catalog, categories, and coupon configuration are read to generate the checkout audit.
  • Local dashboard settings
    Preview mode, low-stock threshold, hidden-check preferences, and owner notes may be saved in browser localStorage.

What The App Does Not Do

Excluded Data Handling

  • No shopper PII capture, checkout field collection, or payment data storage.
  • No remote database, webhook ingestion, Redis, or always-on backend runtime in this repository deployment.
  • No cross-device synchronization for merchant notes or settings.
