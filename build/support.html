Before Contacting Support

Recommended Details

  • Store ID and app page URL
    Include the Ecwid store ID and the public app URL where the issue was observed.
  • What you expected versus what happened
    Describe the audit view, action list, or preview behavior that looks incorrect.
  • Relevant screenshots
    Attach dashboard screenshots with sensitive merchant data removed where practical.

Service Notes

Current Boundaries

  • This app runs as a static admin iframe and reads Ecwid data directly at runtime.
  • Merchant notes and thresholds stay in browser localStorage unless a future version adds remote storage.
  • Support cannot recover browser-local settings that were cleared from the merchant browser.
